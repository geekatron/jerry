# Jerry Claude Skill

A Claude Code Plugin with a Skill.

## Overview

This plugin provides a skill for Claude Code that can be used to extend Claude's capabilities.

## Installation

```bash
npm install
```

## Building

```bash
npm run build
```

## Usage

Once installed, the skill will be available in Claude Code.

## Development

1. Clone this repository
2. Install dependencies: `npm install`
3. Build: `npm run build`
4. Test your skill with Claude Code

## Structure

```
.
├── src/
│   ├── index.ts      # Main entry point
│   └── skill.ts      # Skill implementation
├── lib/              # Compiled output (generated)
├── package.json
├── tsconfig.json
└── README.md
```
