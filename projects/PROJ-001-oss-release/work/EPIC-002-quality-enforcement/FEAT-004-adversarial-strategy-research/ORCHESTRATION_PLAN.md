### 6.3 Escalation Protocol

When a creator-critic cycle exhausts 3 iterations without achieving >= 0.92:

1. **Document** the quality scores for all 3 iterations
2. **Summarize** the persistent quality gaps
3. **Present** to user with options:
   a. Accept current quality score and proceed
   b. Provide guidance for targeted revision
   c. Reassign to different creator agent
   d. Descope the deliverable
4. **Wait** for user decision (P-020: User Authority)
5. **Record** the decision in the workflow state
