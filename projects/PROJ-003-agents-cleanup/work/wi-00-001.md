### Key Findings
- Plugin manifest MUST be `plugin.json` (not `manifest.json`)
- SKILL.md requires YAML frontmatter with `name`, `description`, `version`
- Agent files require `name`, `description`, `model`, `tools` in frontmatter
- hooks.json uses `${CLAUDE_PLUGIN_ROOT}` for portable paths
