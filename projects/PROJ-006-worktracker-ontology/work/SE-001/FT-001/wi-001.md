## Design Deliverables

### 1. Entity Hierarchy

```
WorkItem (abstract)
├── StrategicItem (abstract)
│   ├── Epic
│   ├── Feature
│   └── Capability
├── DeliveryItem (abstract)
│   ├── Story
│   ├── Task
│   ├── Spike
│   └── Enabler
└── DefectItem (abstract)
    ├── Bug
    └── Defect
```

### 2. Entity Schema Template

```yaml
# Entity: {EntityName}
extends: {ParentEntity}
description: "{Description}"

properties:
  # Core (from WorkItem)
  id:
    type: WorkItemId
    required: true
    immutable: true
  title:
    type: string
    required: true
    constraints:
      minLength: 1
      maxLength: 500
  # ... entity-specific properties

behaviors:
  - create
  - update
  - transition
  - link
  - comment

invariants:
  - "title cannot be empty"
  - "status must be valid for entity type"
```
