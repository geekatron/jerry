Consume and analyze major security threat frameworks: MITRE ATT&CK (Enterprise, ATLAS, Mobile), OWASP (LLM Top 10, API Security, Web Top 10, Agentic Top 10), and NIST (AI RMF, CSF 2.0, SP 800-53). Map all frameworks to agentic system attack surfaces.
