## Summary

Conduct trade studies for key feature decisions: (a) marketplace model (curated vs open with verification), (b) multi-model approach (abstraction layer vs model-specific profiles), (c) progressive governance tiers (QuickStart/Team/Enterprise), (d) supply chain verification scope (MCP-only vs full dependency chain). Each study includes options, criteria, evidence from ST-061 and security research, and recommendation.

## Acceptance Criteria

1. Trade study for each of the 4 key decisions
2. Each study includes >= 3 options with criteria
3. Evidence sourced from ST-061 and PROJ-008 security artifacts
4. Clear recommendation with rationale per study
5. Security implications assessed for each option
