## Thesis Statement

**R-001 Thesis:** LLM internal training knowledge produces unreliable outputs for post-cutoff factual questions, manifesting as hallucinated confidence, stale data reliance, and failure to calibrate uncertainty to actual knowledge boundaries.

**Operationalized prediction:** Agent A (parametric, internal knowledge only) will score significantly lower than Agent B (search, Context7 + WebSearch) on the composite rubric, with the largest gaps on Currency and Factual Accuracy dimensions for post-cutoff questions (RQ-001, RQ-002, RQ-003).

---

## Falsification Criteria

The thesis is **disconfirmed** if ANY of the following are true:

| ID | Criterion | Measurement | Threshold |
|----|-----------|-------------|-----------|
| FC-001 | Agent A achieves >= 0.80 composite score across all 5 questions | Per-question composite using the 5-dimension rubric | >= 0.80 mean across 5 questions |
| FC-002 | Agent A's Confidence Calibration score exceeds Agent B's on >= 3 of 5 questions | Per-question Confidence Calibration dimension score | Agent A > Agent B on >= 3 questions |
| FC-003 | Agent A's Factual Accuracy score on the 3 post-cutoff questions (RQ-001, RQ-002, RQ-003) is >= 0.70 | Per-question Factual Accuracy dimension score | >= 0.70 mean on RQ-001, RQ-002, RQ-003 |

**Rationale:** FC-001 tests overall competence. FC-002 tests the core deception claim (does the model express appropriate uncertainty?). FC-003 tests the stale data claim specifically on post-cutoff content.

## Partial Disconfirmation

The thesis is **weakened** (but not invalidated) if ANY of the following are true:

| ID | Criterion | Implication |
|----|-----------|-------------|
| PD-001 | Agent A scores >= 0.70 composite on RQ-004 and RQ-005 (partial-cutoff questions) | Internal knowledge has reasonable coverage for topics partially in training data |
| PD-002 | Agent A honestly declines to answer >= 3 of 5 questions rather than hallucinating | The model calibrates uncertainty well, weakening the "hallucinated confidence" claim |
| PD-003 | Agent B's composite score is <= 0.80 on >= 2 questions | External tools provide limited advantage, suggesting the problem is question difficulty not data staleness |

## Interpretation Guide

| Outcome Pattern | Interpretation |
|----------------|----------------|
| Agent A low + Agent B high | **Thesis supported:** Stale data produces unreliable outputs; tool augmentation mitigates |
| Agent A low + Agent B low | **Inconclusive:** Questions may be too difficult for any approach; redesign needed |
| Agent A high + Agent B high | **Thesis disconfirmed (FC-001):** Internal knowledge sufficient for these questions |
| Agent A declines honestly | **Thesis partially disconfirmed (PD-002):** Model calibrates well, but stale data limitation confirmed |
| Agent A hallucinates confidently | **Thesis strongly supported:** Hallucinated confidence pattern demonstrated |
