## Summary

Create four markdown templates, one per Diataxis quadrant, that writer agents use as structural scaffolding when producing documentation. Each template encodes the quadrant's structural patterns, required sections, and content placeholders.

**Technical Scope:**
- Tutorial template (learning-oriented, step-by-step structure)
- How-to template (task-oriented, goal-focused structure)
- Reference template (information-oriented, comprehensive coverage structure)
- Explanation template (understanding-oriented, conceptual depth structure)

---

## Problem Statement

Writer agents need structural templates to produce consistently formatted output. Without templates, each invocation would produce differently structured documents, making documentation quality unpredictable and hard to audit.

---

## Business Value

Templates ensure:
- Consistent document structure across all outputs
- Correct section ordering per quadrant requirements
- Placeholder guidance for agent content generation
- Auditable structural compliance

### Features Unlocked

- FEAT-013-001: Core Writer Agents (agents use templates for output)

---

## Technical Approach

1. Design template structure based on EN-013-001 research and EN-013-002 standards
2. Create tutorial-template.md with step-by-step learning structure
3. Create howto-template.md with goal-focused task structure
4. Create reference-template.md with comprehensive information structure
5. Create explanation-template.md with conceptual depth structure
6. Place templates in skills/diataxis/templates/
