## Summary

Build a Diataxis documentation framework skill for Jerry that enables structured documentation authoring across the four Diataxis quadrants: tutorials, how-to guides, reference, and explanation. The skill includes writer agents for each quadrant, a classifier agent, an auditor agent, per-quadrant templates, and full routing integration.

**Key Objectives:**
- Implement 4 writer agents (one per Diataxis quadrant) with governance YAML
- Implement classifier and auditor agents for document classification and quality assessment
- Create per-quadrant markdown templates following Diataxis standards
- Register the /diataxis skill in CLAUDE.md, AGENTS.md, and trigger map
- Validate all agents via governance schema and adversarial quality review

---

## Business Outcome Hypothesis

**We believe that** providing a structured Diataxis skill with specialized writer agents and quality gates

**Will result in** consistently high-quality documentation that matches reader intent across all four knowledge dimensions

**We will know we have succeeded when** the skill produces documentation that passes adversarial review >= 0.95 and improves 2+ existing Jerry docs
