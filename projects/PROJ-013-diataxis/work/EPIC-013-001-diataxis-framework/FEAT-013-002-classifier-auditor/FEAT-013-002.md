## Summary

Implement two supporting agents for the Diataxis skill: a classifier agent that determines which Diataxis quadrant a document belongs to, and an auditor agent that evaluates existing documentation against Diataxis standards and identifies quadrant violations.

**Value Proposition:**
- Classifier enables automatic routing of documentation requests to the correct writer agent
- Auditor enables systematic quality assessment of existing documentation against Diataxis principles
- Together they close the feedback loop for continuous documentation improvement

---

## Benefit Hypothesis

**We believe that** classifier and auditor agents will enable both proactive (classify-then-write) and reactive (audit-then-improve) documentation workflows

**Will result in** a complete Diataxis skill that handles the full documentation lifecycle

**We will know we have succeeded when** the classifier correctly routes documents to quadrants and the auditor identifies actionable violations in existing docs
