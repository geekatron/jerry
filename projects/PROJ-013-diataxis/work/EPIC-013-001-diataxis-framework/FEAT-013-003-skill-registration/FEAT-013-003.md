## Summary

Register the /diataxis skill in the Jerry framework by updating CLAUDE.md, AGENTS.md, and the trigger map in mandatory-skill-usage.md. This ensures the skill is discoverable, routable, and invocable through standard Jerry mechanisms.

**Value Proposition:**
- Skill becomes invocable via `/diataxis` slash command
- Trigger map enables proactive invocation per H-22
- Full framework integration ensures consistent behavior with other skills

---

## Benefit Hypothesis

**We believe that** proper skill registration and routing integration

**Will result in** the Diataxis skill being automatically invoked when documentation-related keywords are detected

**We will know we have succeeded when** `/diataxis` triggers correctly and keyword-based routing matches documentation requests to the correct agent
