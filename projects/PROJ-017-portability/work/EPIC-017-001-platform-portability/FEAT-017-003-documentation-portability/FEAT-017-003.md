## Summary

Replace Unix-specific path references in documentation and docstrings with cross-platform alternatives or platform-neutral examples.

**Value Proposition:**
- Documentation examples work when copy-pasted on any platform
- Docstrings use `tempfile.gettempdir()` references instead of hardcoded `/tmp`

---

## Acceptance Criteria

### Definition of Done

- [ ] All child bugs resolved and verified
- [ ] No hardcoded `/tmp` paths in docstrings or user-facing documentation
- [ ] Example paths use platform-neutral notation or show both Unix/Windows variants
