## Overview

This document provides entrance and exit criteria checklists for NASA technical reviews. Each checklist is derived from NPR 7123.1D Appendix G and NASA SWEHB 7.9.

**Usage:** Copy the appropriate checklist into a review package and evaluate each criterion.

## Checklist Legend

| Symbol | Meaning |
|--------|---------|
| ⬜ | Not evaluated |
| ✅ | Criterion met (GREEN) |
| ⚠️ | Criterion partially met (YELLOW) |
| ❌ | Criterion not met (RED) |
| N/A | Not applicable |

---

## Usage Instructions

1. **Copy** the appropriate checklist for your review type
2. **Evaluate** each criterion against project artifacts
3. **Record** evidence reference for each evaluated criterion
4. **Mark** status using the legend above
5. **Document** notes for any YELLOW or RED criteria
6. **Calculate** overall readiness percentage
7. **Determine** recommendation (Ready/Not Ready/Conditional)
